# AI-сервис для анализа отзывов о товарах на маркетплейсах, который помогает продавцу выявлять основные причины негативных отзывов и отслеживать их динамику во времени. Сервис классифицирует отзывы по типам проблем и с помощью RAG объясняет, в чем именно заключается проблема, подтверждая выводы примерами из отзывов.
